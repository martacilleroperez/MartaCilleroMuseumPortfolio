Interactive game for runners with the use of IMU sensors

The aim of the project focuses on developing an IMU (Internal measurement unit) con- trolled running game to involve and invite more young people into the sport of running, as well as to create a possibility for people recovering from lower-body trauma. In such a rehabilitation scenario, the system could act as a distraction factor against pain. The system uses two IMUs, one on the head and one on the leg, to control a real-time game that provides immediate feedback based on the runner’s motion parameters. The key parameters were extracted from the IMU data using custom algorithms and visualized through drifferent game elements.
